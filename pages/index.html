Sobre o Projeto

Este projeto é uma aplicação CRUD (Create, Read, Update, Delete) desenvolvida utilizando JavaScript, HTML e CSS com o framework Bulma. A aplicação permite a criação, leitura, atualização e exclusão de registros de funcionários, e armazena as informações localmente no navegador usando a API localStorage.

Tecnologias Utilizadas

  • JavaScript: Linguagem de programação utilizada para a lógica de negócios e manipulação do DOM.
  • HTML: Linguagem de marcação utilizada para estruturar a aplicação.
  • CSS: Linguagem de estilo utilizada para a aparência da aplicação.
  • Bulma: Framework CSS que fornece uma base de estilos responsivos e modernos.
  • localStorage: API Web Storage que armazena dados no navegador de forma persistente.

O que é localStorage?

localStorage é uma API Web Storage que permite armazenar dados localmente no navegador do usuário. Os dados são armazenados em pares de chave-valor e permanecem disponíveis mesmo após o navegador ser fechado e reaberto. Isso é útil para manter informações entre sessões, como dados de login ou preferências do usuário.

  • Armazenamento persistente: Os dados permanecem armazenados até que sejam explicitamente removidos.
  • Capacidade de armazenamento: Aproximadamente 5-10 MB dependendo do navegador.
  • Fácil de usar: Métodos simples para armazenar e recuperar dados: localStorage.setItem(key, value) e localStorage.getItem(key).

Funcionalidades do CRUD

O sistema CRUD implementado neste projeto inclui as seguintes funcionalidades:

  • Criar: Adicionar novos registros de funcionários.
  • Leitura: Visualizar a lista de funcionários cadastrados.
  • Atualizar: Editar informações de funcionários existentes.
  • Excluir: Remover registros de funcionários.

Acessar o Projeto

Para acessar a aplicação, você pode fazer o login com um usuário e senha predefinidos. Após o login, você será direcionado para a página principal onde poderá gerenciar os logins de usuários.
